The input fields consists of:

  • the Axis-Angle parameters: Theta, the angle of rotation (in degrees) and (xu, yu, zu), a vector along the axis of rotation
  • the input point (or vector) to be rotated : (x, y, z)

The output fields are:

  • the rotation quaternion q corresponding to the Axis-Angle input data
  • the rotated point or vector (x', y', z')
